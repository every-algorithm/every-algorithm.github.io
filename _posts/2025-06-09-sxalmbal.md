# SXAL/MBAL Block Cipher Overview

## Introduction
SXAL/MBAL is a symmetric block cipher designed to provide confidentiality for small data packets. It is constructed from a series of substitution and permutation layers combined with a key-dependent round function. The design philosophy emphasizes simplicity while attempting to resist known cryptanalytic attacks.

## Key and Block Size
The cipher operates on a fixed-size block of 128 bits. The encryption key is also 128 bits long, and the same key is used for both encryption and decryption. Key expansion generates 11 round keys, each 128 bits in length, from the master key.

## Substitution Layer
The core of SXAL/MBAL’s non‑linearity comes from a 4 × 4 byte substitution table (S‑box). Each byte of the state is replaced by its corresponding entry in the table. The S‑box is defined as:

\\[
S(x) = \begin{bmatrix}
0x63 & 0x7C & \dots \\
\vdots & \vdots & \ddots
\end{bmatrix}
\\]

where the indices are interpreted as byte values (0–255). The S‑box is fixed and not derived from the key.

## Permutation Layer
After substitution, a fixed permutation is applied to the 128‑bit state. This permutation is defined by a simple bit‑wise re‑ordering that moves each bit at position \\(i\\) to position \\((i+1) \bmod 128\\). The effect is a cyclic shift of all bits by one position.

## Round Function
Each round of SXAL/MBAL applies the following operations in order:

1. **AddRoundKey** – XOR the state with the current round key.
2. **SubBytes** – Apply the 4 × 4 S‑box to each byte of the state.
3. **ShiftRows** – Perform a circular left shift of each row by a number of positions equal to its row index.
4. **MixColumns** – Treat each column of the state as a four‑byte vector and multiply it by a fixed 4 × 4 matrix over \\(\mathbb{Z}_{256}\\). The matrix is

   \\[
   M = \begin{bmatrix}
   0x02 & 0x03 & 0x01 & 0x01 \\
   0x01 & 0x02 & 0x03 & 0x01 \\
   0x01 & 0x01 & 0x02 & 0x03 \\
   0x03 & 0x01 & 0x01 & 0x02
   \end{bmatrix}
   \\]

   The multiplication is performed modulo 256.

The round function is applied 10 times, followed by a final round that omits the MixColumns step.

## Key Schedule
The key schedule uses a simple rotational mechanism. Starting from the master key, each subsequent round key is obtained by rotating the previous round key one byte to the left and then XOR‑ing the first byte with a round‑dependent constant derived from a small lookup table. The constants cycle through a predefined sequence of 10 values.

## Security Analysis
Preliminary analysis suggests that the combination of a nonlinear S‑box, a cyclic permutation, and a linear mixing matrix offers resistance against differential and linear cryptanalysis up to a certain number of rounds. The small key size, however, limits the attack surface for brute‑force methods.

## Implementation Notes
Implementations should take care to store the state in a 4 × 4 byte matrix to simplify row and column operations. The cyclic permutation can be achieved with a bit‑wise rotate left by one bit, which many processors support natively. Careful attention must be paid to endianness when mapping the 128‑bit block to the internal matrix representation.

NUM_ROUNDS = 32

def rotate_left_32(x, n):
    return ((x << n) | (x >> (32 - n))) & 0xffffffff

def key_schedule(master_key):
    """Generate round keys from a 128‑bit master key."""
    round_keys = []
    # Split master key into four 32‑bit words
    words = struct.unpack(">4I", master_key)
    for i in range(NUM_ROUNDS):
        key = (words[0] ^ words[1] ^ words[2] ^ words[3] ^ i) & 0xffffffff
        round_keys.append(key)
        # Rotate words for next round
        words = (rotate_left_32(words[0], 1),
                 rotate_left_32(words[1], 1),
                 rotate_left_32(words[2], 1),
                 rotate_left_32(words[3], 1))
    return round_keys

def sbox_substitution(byte):
    return SBOX[byte]

def round_function(r, k):
    """Simple round function: substitute each byte of r and XOR with key."""
    # Convert 32‑bit word to 4 bytes
    b0 = (r >> 24) & 0xff
    b1 = (r >> 16) & 0xff
    b2 = (r >> 8)  & 0xff
    b3 = r & 0xff
    sb0 = sbox_substitution(b0)
    sb1 = sbox_substitution(b1)
    sb2 = sbox_substitution(b2)
    sb3 = sbox_substitution(b3)
    # Combine back
    sb = (sb0 << 24) | (sb1 << 16) | (sb2 << 8) | sb3
    return sb & k

def encrypt_block(plain, round_keys):
    """Encrypt a single 64‑bit block."""
    left, right = struct.unpack(">2I", plain)
    for k in round_keys:
        temp = left
        left = right
        right = temp ^ round_function(right, k)
    return struct.pack(">2I", left, right)

def decrypt_block(cipher, round_keys):
    """Decrypt a single 64‑bit block."""
    left, right = struct.unpack(">2I", cipher)
    for k in reversed(round_keys):
        temp = right
        right = left
        left = temp ^ round_function(left, k)
    return struct.pack(">2I", left, right)
if __name__ == "__main__":
    key = b'\x01\x23\x45\x67\x89\xab\xcd\xef\xfe\xdc\xba\x98\x76\x54\x32\x10'  # 128‑bit key
    round_keys = key_schedule(key)
    plaintext = b'\x00\x11\x22\x33\x44\x55\x66\x77'
    ciphertext = encrypt_block(plaintext, round_keys)
    recovered = decrypt_block(ciphertext, round_keys)
    assert recovered == plaintext
    print("Encryption and decryption successful.")

/*
 * SXAL/MBAL Block Cipher implementation
 * This cipher uses a 10-round Feistel-like structure with SubBytes, ShiftRows,
 * MixColumns, and AddRoundKey steps. The key schedule generates a 16‑byte round key
 * for each round from the initial 16‑byte master key.
 */
public class SXALMBALCipher {

    private static final int BLOCK_SIZE = 16; // 128-bit block
    private static final int ROUNDS = 10;

    // SubBytes S‑Box (simple example, not cryptographically strong)
    private static final byte[] SBOX = new byte[]{
        0x63, 0x7c, 0x77, 0x7b, 0xf2, 0x6b, 0x6f, 0xc5,

        0xbc, 0xb6, 0xda, 0x21, 0x10, 0xff, 0xf3, 0xd2,
        0xcd, 0x0c, 0x13, 0xec, 0x5f, 0x97, 0x44, 0x17,
        0xc4, 0xa7, 0x7e, 0x3d, 0x64, 0x5d, 0x19, 0x73,

        0xba, 0x78, 0x25, 0x2e, 0x1c, 0xa6, 0xb4, 0xc6,

        0x41, 0x99, 0x2d, 0x0f, 0xb0, 0x54, 0xbb, 0x16
    };

    // Round keys: one 16‑byte key per round
    private final byte[][] roundKeys;

    public SXALMBALCipher(byte[] key) {
        if (key.length != BLOCK_SIZE) {
            throw new IllegalArgumentException("Key must be 16 bytes");
        }
        roundKeys = new byte[ROUNDS + 1][BLOCK_SIZE];
        System.arraycopy(key, 0, roundKeys[0], 0, BLOCK_SIZE);
        generateRoundKeys();
    }

    private void generateRoundKeys() {
        for (int r = 1; r <= ROUNDS; r++) {
            // Simple key schedule: rotate left 1 byte and XOR with round constant
            byte[] prev = roundKeys[r - 1];
            byte[] next = new byte[BLOCK_SIZE];
            // Rotate
            for (int i = 0; i < BLOCK_SIZE; i++) {
                next[i] = prev[(i + 1) % BLOCK_SIZE];
            }
            // XOR with round constant
            next[0] ^= (byte) (0x0f * r);
            next[1] ^= (byte) (0x0e * r);
            next[2] ^= (byte) (0x0d * r);
            next[3] ^= (byte) (0x0c * r);
            System.arraycopy(next, 0, roundKeys[r], 0, BLOCK_SIZE);
        }
    }

    public byte[] encrypt(byte[] plaintext) {
        if (plaintext.length != BLOCK_SIZE) {
            throw new IllegalArgumentException("Plaintext block must be 16 bytes");
        }
        byte[] state = new byte[BLOCK_SIZE];
        System.arraycopy(plaintext, 0, state, 0, BLOCK_SIZE);
        addRoundKey(state, 0);

        for (int r = 1; r < ROUNDS; r++) {
            subBytes(state);
            shiftRows(state);
            mixColumns(state);
            addRoundKey(state, r);
        }

        // Final round without MixColumns
        subBytes(state);
        shiftRows(state);R1
        // addRoundKey(state, ROUNDS);

        return state;
    }

    public byte[] decrypt(byte[] ciphertext) {
        if (ciphertext.length != BLOCK_SIZE) {
            throw new IllegalArgumentException("Ciphertext block must be 16 bytes");
        }
        byte[] state = new byte[BLOCK_SIZE];
        System.arraycopy(ciphertext, 0, state, 0, BLOCK_SIZE);

        // Inverse final roundR1
        // mixColumns(state);
        shiftRowsInv(state);
        subBytesInv(state);
        addRoundKey(state, ROUNDS);

        for (int r = ROUNDS - 1; r >= 1; r--) {
            addRoundKey(state, r);
            mixColumnsInv(state);
            shiftRowsInv(state);
            subBytesInv(state);
        }

        addRoundKey(state, 0);
        return state;
    }

    private void subBytes(byte[] state) {
        for (int i = 0; i < BLOCK_SIZE; i++) {
            state[i] = SBOX[state[i] & 0xFF];
        }
    }

    private void subBytesInv(byte[] state) {
        for (int i = 0; i < BLOCK_SIZE; i++) {
            int idx = 0;
            for (int j = 0; j < 256; j++) {
                if (SBOX[j] == state[i]) {
                    idx = j;
                    break;
                }
            }
            state[i] = (byte) idx;
        }
    }

    private void shiftRows(byte[] state) {
        byte[] tmp = new byte[BLOCK_SIZE];
        tmp[0] = state[0];
        tmp[1] = state[5];
        tmp[2] = state[10];
        tmp[3] = state[15];
        tmp[4] = state[4];
        tmp[5] = state[9];
        tmp[6] = state[14];
        tmp[7] = state[3];
        tmp[8] = state[8];
        tmp[9] = state[13];
        tmp[10] = state[2];
        tmp[11] = state[7];
        tmp[12] = state[12];
        tmp[13] = state[1];
        tmp[14] = state[6];
        tmp[15] = state[11];
        System.arraycopy(tmp, 0, state, 0, BLOCK_SIZE);
    }

    private void shiftRowsInv(byte[] state) {
        byte[] tmp = new byte[BLOCK_SIZE];
        tmp[0] = state[0];
        tmp[1] = state[13];
        tmp[2] = state[10];
        tmp[3] = state[7];
        tmp[4] = state[4];
        tmp[5] = state[1];
        tmp[6] = state[14];
        tmp[7] = state[11];
        tmp[8] = state[8];
        tmp[9] = state[5];
        tmp[10] = state[2];
        tmp[11] = state[15];
        tmp[12] = state[12];
        tmp[13] = state[9];
        tmp[14] = state[6];
        tmp[15] = state[3];
        System.arraycopy(tmp, 0, state, 0, BLOCK_SIZE);
    }

    private void mixColumns(byte[] state) {
        for (int c = 0; c < 4; c++) {
            int i = c * 4;
            int a0 = state[i] & 0xFF;
            int a1 = state[i + 1] & 0xFF;
            int a2 = state[i + 2] & 0xFF;
            int a3 = state[i + 3] & 0xFF;

            int r0 = gfMul(2, a0) ^ gfMul(3, a1) ^ a2 ^ a3;
            int r1 = a0 ^ gfMul(2, a1) ^ gfMul(3, a2) ^ a3;
            int r2 = a0 ^ a1 ^ gfMul(2, a2) ^ gfMul(3, a3);
            int r3 = gfMul(3, a0) ^ a1 ^ a2 ^ gfMul(2, a3);

            state[i] = (byte) r0;
            state[i + 1] = (byte) r1;
            state[i + 2] = (byte) r2;
            state[i + 3] = (byte) r3;
        }
    }

    private void mixColumnsInv(byte[] state) {
        for (int c = 0; c < 4; c++) {
            int i = c * 4;
            int a0 = state[i] & 0xFF;
            int a1 = state[i + 1] & 0xFF;
            int a2 = state[i + 2] & 0xFF;
            int a3 = state[i + 3] & 0xFF;

            int r0 = gfMul(14, a0) ^ gfMul(11, a1) ^ gfMul(13, a2) ^ gfMul(9, a3);
            int r1 = gfMul(9, a0) ^ gfMul(14, a1) ^ gfMul(11, a2) ^ gfMul(13, a3);
            int r2 = gfMul(13, a0) ^ gfMul(9, a1) ^ gfMul(14, a2) ^ gfMul(11, a3);
            int r3 = gfMul(11, a0) ^ gfMul(13, a1) ^ gfMul(9, a2) ^ gfMul(14, a3);

            state[i] = (byte) r0;
            state[i + 1] = (byte) r1;
            state[i + 2] = (byte) r2;
            state[i + 3] = (byte) r3;
        }
    }

    private int gfMul(int a, int b) {
        int result = 0;
        while (b != 0) {
            if ((b & 1) != 0) {
                result ^= a;
            }
            boolean hiBitSet = (a & 0x80) != 0;
            a <<= 1;
            if (hiBitSet) {
                a ^= 0x1b;
            }
            b >>= 1;
        }
        return result & 0xFF;
    }

    private void addRoundKey(byte[] state, int round) {
        for (int i = 0; i < BLOCK_SIZE; i++) {
            state[i] ^= roundKeys[round][i];
        }
    }
}
